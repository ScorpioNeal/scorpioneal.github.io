Algorithm LeetCode 155 Min Stack

Posted on 2019-12-14 | In Algorithm |

题目

Design a stack that supports push, pop, top, and retrieving the minimum element in constant time.

push(x) – Push element x onto stack.
pop() – Removes the element on top of the stack.
top() – Get the top element.
getMin() – Retrieve the minimum element in the stack.

Example:

MinStack minStack = new MinStack();
minStack.push(-2);
minStack.push(0);
minStack.push(-3);
minStack.getMin(); –> Returns -3.
minStack.pop();
minStack.top(); –> Returns 0.
minStack.getMin(); –> Returns -2.

思路

每次push的时候, 如果当前是最小值, 则把之前的min值放进去也就是把倒数第二min的值放进去
每次pop的时候，如果pop是最小值，min则指向之前记录的倒数第二值作为pop后的最小值
如果pop的时候，不是最小值，就直接pop

也就是如果push的时候是最小值，就push 2次，push一个最小值，push一个倒数第二, 如果不是就push一次
pop的时候，如果是最小值，就pop 2次 min记录第二次pop的值作为后面的最小值, 如果不是,就pop一次
